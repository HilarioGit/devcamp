FAQ- Perguntas Frequentes:

  • O que é o Devcamp?

    É um programa de imersão tecnológica com foco para DevOps e Gestão de Infraestrutura TI, direcionada a profissionais e estudantes da área que desejam se actualizar e aprimorar suas habilidades num ambiente mais

  • Quem pode participar do Programa Devcamp?

    Profissionais e estudantes de TI.

  • O que eu vou aprender no Programa Devcamp?

    Criar máquina virtuais em infraestrutura de nuvem publica
    Como hospedar a sua aplicação /website num servidor cPanel
    Como instalar, administrar servidores Linux
    Como escolher o frameworks java certo para o seu projecto,e como começar um projecto de desenvolvimento web
    como funcionam o HTML e CSS
    TypScript
    UIKit (biblioteca front-end)
    Git. Como criar, usar e gerenciar repositórios do GitHub
    O poder do SHELLSCRIPT e do protocolo SSH no gerenciamento da sua infraestrutura de computação
    gerenciamento de servidores cpanel.

  • Qual é a metodologia de ensino do Programa Devcamp?

    Metodologia de ensino prática, com atividades individuais e projetos reais.

  • Qual a duração do Programa Devcamp?

    O programa tem duração de 6 horas diárias separados em 2 dias.
